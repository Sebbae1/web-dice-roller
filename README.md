## Description:
A project that generates five numbers with a 6-sided dice.
This reinforces the understanding of JavaScript and its implementation
with HTML and CSS. [How to Create a Dice Roll Game using HTML CSS & JavaScript](https://www.youtube.com/watch?v=M3InbHr0WAc)
## Instructions for Compiling and Executing
The main languages used for this project are HTML, CSS, and JavaScript.
1. Clone the repository to your system.
2. Go to the located repository in your file system; web-dice-roller > static > index.html
3. Select the index.html file and open it on Google Chrome, Microsoft Edge, or Firefox.
